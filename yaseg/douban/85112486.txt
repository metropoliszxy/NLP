Testing Technical Summary-other testing

Exploratory Testing:While testing, do research about product/software and do testing for feature with potential risk. 	Bugs are hard to be found after several testing cycles are finished. Tester do Exploratory testing can use the different way to verify the product&Software. 	More bugs can be found by testing. -----------------------------------------------------------------------------------------Ad hoc testing :beside test case execution, try more user case or flow After verify bug fix, do testing for related feature. To find bugs in potential part. 	Ad hoc testing can find bug more efficiently:  1. Mostly test case cannot cover all the use case  2. Test Case maintenance required much effort. It much depend on the tester's ability. -----------------------------------------------------------------------------------------Monkey testing :Without purpose, randomly testing product/softwareIt is better to be designed as automation testing.  It is often to be used in stability testing.

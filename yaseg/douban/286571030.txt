1. 所有关于时间管理和规划的文章都基于一个前提：你十分希望自己可以有效管理时间并把产出最大化，然后获得成功。这个前提在「竞争如此激烈的社会」好像已经被无条件接受了，只有 loser 才会想无所事事。作为一个经常焦虑的人，我倾向躲避这种文章，虽然其中不乏可取之处。此文适合鄙视成功学和心灵鸡汤却又渴望成功的人阅读。2. 这篇东西发表在 UK 政府网站，不只有理论，还有实践案例，让我们从另外一个角度知道了什么是「服务型政府」。摘录一段：Government should only do what only government can do. If someone else is doing it — link to it. If we can provide resources (like APIs) that will help other people build things — do that. We should concentrate on the irreducible core.3. 如果仔细回想别人给的建议，会发现「多读书」这个建议都是读书人给的。读书越多，表达「读书就是好」这种观点的热情就越强烈。如果一个人喜欢吃苹果就认为其他人也应该吃，我们肯定会觉得此人自以为是，但读书人教人多读书就没人奇怪。听腻了「读书让人更好」的叫嚣，我现在非常谨慎地听取读书人关于读书的建议，更喜欢像本文这种比较中立的反思。读书时享受审美愉悦或许已经足够，不用再去说明自己因此更进步。4. 收集了很多有意思的 Mobile UI, 点击可以看大图。5. 奇人（陈光标）让我发现了一个奇怪现象，我们一听到留取丹心照汗青、做一番功业为了在青史上留名之类的壮志豪情，就觉得可敬可歌，为什么本着在历史上留个慈善之名而行善的，就可鄙起来？历史上很多清流派，没真正有能力做过几件好事，反而为博清名，误了许多大事。纯粹做好事的好官好人善人，是应该连身后名都没想过的，甚至为做好事而有损自身名声，也在所不惜，更不会说什么历史自有公论。但这样纯粹的人与事，太难求了，退而求其次，以名誉做饵，引诱人成为不够纯粹的好人，做些好事，总是好事。行善不为人知，只是个人选择，要追根究底一个人行善的动机够不够纯粹，非常无谓。行善若为积福，若为得到所谓快乐之本，低调得来又有几纯粹？—— 林夕《》
